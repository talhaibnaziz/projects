Rhezo and Destructive Mind

Rhezo has bought a graph with NN nodes and MM edges. He has a destructive mind and often deletes some nodes(and of course its edges to other nodes of the graph).

He will ask you QQ queries. Each query is denoted by an integer XX, meaning that he will delete the node XX. Since he has a destructive mind, he gets satisfied if the the new graph with deleted node has more number of connected components than the original graph.

If satisfied, he wants you to print "Satisfied"(without quotes) and the number of edges that were removed when deleting node XX, else print "Not Satisfied"(without quotes). All queries are different, and the original graph is restored after each query.

Input:

First line contains NN and MM as described in the problem statement. Each of the next MM lines contain 2 integers AiAi and BiBi, meaning that there is an edge between node AA and node BB. Next line contains a single integer QQ, denoting the number of queries. Each of the next QQ lines contain a single integer XX as described in the problem statement.

Output:

For each of the QQ queries, if Rhezo is satisfied(that is the number of connected components of the new graph has increased), print "Satisfied"(without quotes), and the number of edges that will be deleted after a space. Else print "Not Satisfied".

Constraints:

1≤N,M,Q≤1021≤N,M,Q≤102

1≤Ai,Bi≤N1≤Ai,Bi≤N

1≤X≤N1≤X≤N

5 5
1 2

SAMPLE OUTPUT
Not Satisfied
Satisfied 2

Explanation

For the second query, if we remove node 2, the graph disconnects and 2 connected components are formed. One having only {1} and other having {3,4,5}. Clearly, the number of edges removed are 2.
